# Basic Mastering Chain

tags: mastering, eq, compression, limiting

Signal chain order matters. A typical starting point:

1. **Linear Phase EQ** — Broad corrective moves, cut resonances
2. **Multiband Compression** — Control dynamics per frequency band
3. **Stereo Enhancement** — Widen mids/highs, keep bass mono
4. **Limiter** — Final loudness ceiling, -1dB true peak for streaming

Target loudness:
- Streaming (Spotify/Apple): -14 LUFS integrated
- Club/DJ: -8 to -6 LUFS
- Podcast: -16 LUFS
